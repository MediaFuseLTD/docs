## How to use

1. Download the collection JSON file above.
2. Open Postman and click the **Import** button in the top left.
3. Drag and drop the downloaded JSON file.
4. Once imported, click on the **Chainwire External API** collection folder.
5. Go to the **Authorization** tab for the collection.
6. Ensure the type is set to **Bearer Token**.
7. Paste your `cw_ext_...` API key into the Token field.
8. Save the collection. All requests inside the folder will now automatically inherit this token.

You're ready to start testing!
